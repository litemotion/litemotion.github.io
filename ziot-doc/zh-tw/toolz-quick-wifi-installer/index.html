    自定義的 WiFi 安裝包

    2019-12-02

    預設出廠 WiFi 設定

    從 2019 年 11 月起，所有預安裝好的基盒 Basebox 中都統一預設以下 WiFi 設定:

    設定 預設值
    網路名稱 (也稱為 SSID) ziot
    網路安存密碼 (Network Security Password) litemotion!+SGVsbG8gUm91bnRlcg==,IEkgd2FudCB0byBkYXRlIHlvdS4=

    如果把你的 WiFi 路由器設定成如以上的設定，Basebox 則不用在 WiFi 設定作任何改動。

    預設 WiFi 安裝包

    從版本 2.0.13 或以後，toolz 均包括了一個安裝以上預設設定的 WiFi 安裝包(installer)，名為 base3-wifi。

    建議你不要修改這個預設 WiFi 安裝包 base3-wifi，以便將來有需要時使用它來把 Basebox 設定成預設的 WiFi 設定:。

    複製 預設 WiFi 安裝包

    你可以使用 toolz 的複製安裝包的方法，來產生自己的 WiFi 安裝包，然後修改當中的內容。

    例子

    假設你在一台 WiFi 路由器上設定它的 SSID 為 happy, 和 密碼 為 A366?pm78。

    1. 複製 預設 WiFi 安裝包 到一個新增的的安裝包，你可以定義新的的安裝包名稱為 base3-wifi-happy。

      執行以下命令: 

      tool usb>install base3-wifi -to base3-wifi-happy

    2. 打開新增的的安裝包，修改安裝包中 userconfig.lua 內的 SSID 及 Password 參數。

      原本的 userconfig.lua 內部: 

      WIFI_SSID = "ziot"
WIFI_PWD  = "litemotion!+SGVsbG8gUm91bnRlcg==,IEkgd2FudCB0byBkYXRlIHlvdS4="
SERVER_DOMAIN="zctl-iot-server.eastasia.cloudapp.azure.com:8080"--"192.168.133.176:8080"
COUNTER_ON = true
ACCEL_ON = false

      修改當中的 WIFI_SSID 和 WIFI_PWD 兩行為你的 SSID 和 Password: 

      WIFI_SSID = "happy"
WIFI_PWD  = "A366?pm78"
SERVER_DOMAIN="zctl-iot-server.eastasia.cloudapp.azure.com:8080"--"192.168.133.176:8080"
COUNTER_ON = true
ACCEL_ON = false
