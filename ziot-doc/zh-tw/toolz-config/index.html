    配置 Toolz (Config Toolz)

    2019-11-27

    toolz 的配置檔名為 config.json，位於 tool.exe 的同一個目錄內。本文下方有一個 config.json 的例子，可以參考一下它大概是怎樣。

    Warning

    錯誤地修改 config.json 會令 toolz 無法正常運作。

    Note

    一般情況下，你並不需要手工修改 config.json。

    第一層配置

    config.json 的第一層配置項如下:

    item type description
    name String config 的名稱
    version String config 的 version
    description String config 的描述
    author String config 的作者
    title String tool.exe 程式的 title
    homepage String toolz 的 線上文檔 URL 路徑
    repository Object toolz 的系統更新配置
    workspace Object toolz 的 workspace (工作空間，即本機相關的) 配置
    profiles Array of Object toolz 要連線的服務器 profile 配置
    command Object toolz 的預設指令 (Preset Command) 配置

    repository 配置

    repository 配置定義系統更新 (system update) 的網庫位置及模式。

    repository:{
  url:"http://...",
  checkUpdate:"manual"
}

    repository.url

    String. 定義系統更新的網上程序庫 (網庫) 文件位置。

    repository.checkUpdate

    String. 定義系統更新的模式。
    可用值: manual - 手動模式。用戶使用 search 命令來檢查更新。

    workspace 配置

    toolz 的 workspace 是用戶在其電腦上執行 toolz 的一系列相關的配置。包括:

    • 定義 toolz 的多語言配置
    • 定義 toolz 在電腦上可啟用的文字編輯器
    • 定義 toolz 在電腦上存放 MOS 的位置
    • 定義 toolz 在電腦上存放 flasher 的位置
    • 定義 toolz 在電腦上有關使用 usb 的配置
