    安裝 BASE03F3

    2019-12-04

    本文描述在空白的 Basebox 安裝 BASE03F3 的步驟。(適合初學者)

    如果想了解 BASE03F3 運行在那些 Basebox 型號，參考 Basebox 型號的列表

    需要的物件

    1. 沒有安裝程序的 Basebox
    2. micro USB 數據線
    3. toolz 程式
    4. BASE03F3 的安裝包

    預備動作

    不論你要對多少個 Basebox 執行，預備動作只需在「執行動作」之前做一次。

    1. 用 micro USB 數據線 連接 Basebox 及你的電腦端口

      Note

      以下假設: 你的電腦連接端口是 COM3。而你的預設端口亦是 COM3。

    2. 在 toolz 轉到 usb 模組

      Note

      如果你目前不在 usb 模組，請鍵入命令 cd usb。如果你已在 usb 模組，則略過此步。

    3. 在 toolz usb 模組，檢查是否有 BASE03F3 安裝包

      Note

      在 usb 模組，請鍵入命令 install base3。如果出現下圖內容表示已有安裝包。

    執行動作

    對每個要處理的 Basebox ，執行動作都需做一次。

    1. 用 micro USB 數據線 連接 Basebox 及你的電腦端口

      Note

      如果接上沒有反應，而確定沒有其他程式正常使用該端口，可以重新接上試試。如果接上好幾次仍沒有反應，可能 Basebox 本身有損壞。

    2. 對空白的 Basebox 燒入(flash) MOS。

      Note

      執行以下命令，開啟 Flasher 程式。

      tool usb> flash

      在 Flasher 程式上檢查 Operation頁 的 COM Port 及 Config頁 的檔案路徑。無問題就按 Flash 按鈕。等待至燒錄過程完成。

      如果按 Flash 按鈕後未能啟動燒錄過程，可重按 Flash 按鈕，如果多次仍未能啟動燒錄過程，可能 Basebox 本身有損壞。

    3. Format Basebox 快閃記憶體

      Note

      按下 Basebox 的 RST 按鈕，或者執行預設指令 restart。

    4. 檢查 MOS

      Note

      執行以下命令 (此步為 Optional)

      tool usb> detect mos

    5. 安裝 BASE03F3 安裝包

      Note

      執行以下命令

      tool usb> install base3

    6. 檢查 Base Libraries

      Note

      執行以下命令 (此步為 Optional)

      tool usb> detect lib

    7. 檢查 Basebox Auto Mode

      Note

      執行以下預設指令 (此步為 Optional)

      tool usb> cmd check-mode
