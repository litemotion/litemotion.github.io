    Machines Supported

    2019-11-20

    At the time of writing, we have plan to support at least one model of the following types of machines with the Basebox.

    Plan

    Class 機類 訊號事件 實現情況 備註
    Injection Molding 注塑機 壓煉機的電動開模事件 ✔
    Injection Machine Robot Arm 機械臂 壓煉機通知而開始啟動的事件 ✔
    Automatic Spray Machines 自動噴油機 噴油機的電動關門事件 ✔
    Automatic Spray Machines 自動噴油機 噴油機的噴咀開關事件 ✔
    Blow Molding 吹塑機 吹塑機的電動啟動切刀事件 ✔
    Pad Printer 移印機 移印機氣缸啟動中軸推出事件 ✔
    Doll Hair Rooting Machine 車髮機
    Vacuum Forming Machine 真空成型吸索機, 吸索機 吸索機的電動啟動切刀事件
    Sewing Machine 電動衣車
