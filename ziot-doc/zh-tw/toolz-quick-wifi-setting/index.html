    設定 WiFi

    2019-12-04

    需要的物件

    1. 已經安裝好程序的 Basebox
    2. micro USB 數據線
    3. toolz 程式
    4. 想要 Basebox 連接上的 WiFi 連線的安裝包

    預備動作

    不論你要對多少個 Basebox 執行，預備動作只需在「執行動作」之前做一次。

    1. 用 micro USB 數據線 連接 Basebox 及你的電腦端口

      Note

      以下假設: 你的電腦連接端口是 COM3。而你的預設端口亦是 COM3。

    2. 在 toolz 轉到 usb 模組

      Note

      如果你目前不在 usb 模組，請鍵入命令 cd usb。如果你已在 usb 模組，則略過此步。

    執行動作

    對每個要處理的 Basebox ，執行動作都需做一次。

    1. 在 toolz 確定 Basebox 已在 非自動執行模式 (stop-auto mode)

      Note

      如果 Basebox 已在 非自動執行模式 ，則略過此步。

      執行預設指令 stop-auto

      tool usb> cmd stop-auto

    2. 在 toolz 執行安裝 WiFi 相關安裝包

      Note

      以下假設: 你的 WiFi 相關安裝包名稱是 base3-wifi-d1f。安裝包內容是設定 Basebox WiFi SSID 為 d1f。

      執行命令

      tool usb> install base3-wifi-d1f

      如果 toolz 找到該安裝包 (base3-wifi-d1)，會把安裝包內的描述及安裝腳本的步驟顯示出來，並且詢問你是否確定安裝。確定請輸入 y，取消請輸入 n。

    3. 在 toolz 設定 Basebox 為 自動執行模式 (auto mode)

      執行預設指令 auto

      tool usb> cmd auto
    動畫: toolz 設定 WiFi
